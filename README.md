# Kyuwon Kim

**Principal Software Engineer & Computer Vision/Imaging Researcher at Samsung Electronics**

I work at the pixel level — image and video quality, codec and compression analysis, on-device AI, and 3D scene reconstruction. I hold **15+ issued international patents**, with many features actively shipping in Samsung Galaxy devices, and have published at **BMVC** and **IEEE Transactions on Consumer Electronics**.

### 🚀 Featured Work

#### [🔬 Q1Net](https://github.com/chammoru/Q1Net)
Block-wise, confidence-aware CNN that predicts the compression quality level (e.g., a JPEG quality factor) of an image in real time. Official TensorFlow implementation of the **BMVC 2021** paper.

#### [🛠️ Q1View](https://github.com/chammoru/Q1View)
High-performance Windows toolkit for inspecting the pixels ordinary players hide: raw YUV/RGB buffers, codec output, HEIF/AVIF stills, and frame-by-frame comparison.
